# 写真共有アプリ

## 概要

写真共有アプリ<br />
写真共有アプリのポートフォリオです。<br />
会員登録なしでも、世界中のユーザーが投稿した写真を閲覧することが出来ます。<br />
会員登録することでユーザー自身で写真を共有したり、<br />
他のユーザーが投稿した写真にいいねやコメント投稿を行うことが出来るようになります。<br />

## 使用技術

-   CSS3
-   Sass
-   JavaScript
-   Vue.js
-   PHP7.4.3

-   写真閲覧
-   会員登録
-   ログイン
-   写真投稿
-   いいね
-   コメント投稿
-   エラーページ
-   など、随時追加予定。。。
